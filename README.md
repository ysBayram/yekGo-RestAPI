# yekGo-RestAPI
yekGo-RestAPI create Rest API for SqLiteDB with Gorm and Gin framework

## Quick Start

```bash
git clone https://github.com/ysBayram/yekGo-RestAPI.git
cd ./yekGo-RestAPI
go run main.go
```

API running on http://localhost:4141

```bash
curl http://localhost:4141/ogrenci/1
```
